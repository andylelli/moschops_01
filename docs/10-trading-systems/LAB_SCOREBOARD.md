## Stage Definitions
1. Discovery: design and validation framework still being hardened.
2. Paper-ready: all research gates pass and paper deployment criteria met.
3. Paper-running: live paper observation in progress.
4. Micro-live-ready: paper evidence complete and approved.
5. Micro-live-running: controlled capital pilot active.
6. Scale-ready: sustained micro-live pass and risk sign-off complete.

## Scoring Guidance
1. A gate can be PASS only with linked evidence artifacts.
2. Any missing critical evidence forces gate status to Not ready.
3. Any invalid run forces promotion decision to REJECT until remediated.

## Review Rhythm
1. Update this scoreboard after each gate review.
2. Keep Last Review date current.
3. Add links to key review artifacts in each system folder index.
