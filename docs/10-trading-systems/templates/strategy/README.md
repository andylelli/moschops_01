# Trading Systems Template Documentation Set

## Purpose
1. Provide a complete, reusable documentation pack for any future trading system (TS-00x).
2. Enforce no-backtest-bending governance and consistent engineering evidence.
3. Standardize strategy design, validation, promotion, and operations documentation.

## How To Use This Pack
1. Copy the full set into a new strategy folder, for example docs/10-trading-systems/TS-002.
2. Rename files by removing _TEMPLATE and replacing TS-XXX with the real system ID.
3. Fill required docs first, then optional docs.
4. Do not promote any system unless the completion checklist and promotion gates are fully satisfied.
